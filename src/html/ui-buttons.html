Buttons

Default Buttons

Use the button classes on an <a>, <button>, or <input> element.

<button type="button" class="btn btn-primary waves-effect width-md">Primary</button>

Button-Width

Create buttons with minimum width by adding add .width-xs, .width-sm, .width-md or .width-lg.

Button-Sizes

Add .btn-lg, .btn-sm, or .btn-xs for additional sizes.

Button-Disabled

Add the disabled attribute to <button> buttons.
